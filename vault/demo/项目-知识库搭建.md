# 项目-知识库搭建

## 项目概述

构建一个基于 Karpathy 方法论的个人知识库系统，整合 Obsidian、ChromaDB、FastAPI 和 React，实现笔记的语义搜索和 AI 问答功能。

## 目标

1. 建立统一的笔记管理流程
2. 实现本地 LLM 驱动的 RAG 问答
3. 支持双向链接和知识图谱可视化
4. 打造无缝的写作和研究体验

## 技术架构

### 技术栈
- **笔记存储**：Obsidian Vault（本地 Markdown 文件）
- **后端框架**：FastAPI + Python 3.12
- **前端框架**：React + Vite + TypeScript + Tailwind CSS
- **向量数据库**：ChromaDB（轻量级、易部署）
- **LLM**：Ollama（本地运行，数据不外泄）

详见 [[docs/design/architecture.md|架构设计文档]]
